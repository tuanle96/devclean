## [0.1.0] - 2026-07-11

### Added

- Read-only scanning with terminal, JSON, and HTML output.
- Conservative and comprehensive cleanup profiles.
- Evidence-based Rust target, Node, framework, build, and test-cache detection.
- Allowlisted global package and development-tool cache cleanup.
- Docker cleanup that preserves volumes.
- Candidate revalidation, symlink rejection, mount-boundary protection, and allocated-size accounting.
- Companion Codex skill and open-source project automation.
