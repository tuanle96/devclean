# Support

- Use [GitHub Discussions](https://github.com/tuanle96/devclean/discussions) for installation, usage, and policy questions.
- Use [GitHub Issues](https://github.com/tuanle96/devclean/issues) for reproducible bugs and scoped feature requests.
- Use [private vulnerability reporting](https://github.com/tuanle96/devclean/security/advisories/new) for security-sensitive behavior.

Please include the output of `devclean doctor`, the operating system, the `devclean` version, and a redacted command. Never post secrets, private paths, database contents, or unreviewed scan reports publicly.
