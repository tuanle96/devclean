# devclean v0.1.0

The first public release of `devclean`, a safety-first Rust CLI for reclaiming disk space used by rebuildable development artifacts.

## Highlights

- Audit-first terminal, JSON, and HTML reports.
- Conservative and comprehensive cleanup modes.
- Evidence-based Rust `target` detection.
- Optional global package-cache and Docker cleanup.
- Hard protection for symlinks, VCS metadata, backups, PostgreSQL paths, and Docker volumes.
- Companion `dev-disk-cleaner` Codex skill.

Download the archive for your platform and verify it with `SHA256SUMS`.
